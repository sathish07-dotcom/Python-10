Inheritance in Python

Inheritance is a feature of **Object-Oriented Programming (OOP)** that allows one class (**child class**) to inherit the properties and methods of another class (**parent class**).

Benefits of Inheritance

  • Code Reusability
  • Increases Code Organization
  • Promotes Modularity
  • Supports Hierarchical Classification

    • Basic Syntax

    class Parent:
    def method1(self):
    print("Parent method")

    class Child(Parent):
    def method2(self):
    print("Child method")

    obj = Child()
    obj.method1() # Inherited from Parent
    obj.method2()

    Types of Inheritance

    Single Inheritance

  • One child class inherits from one parent class.

    • class A:
    pass

    class B(A):
    pass

    Multiple Inheritance

  • A child class inherits from multiple parent classes.

    • class A:
    pass

    class B:
    pass

    class C(A, B):
    pass

    Multilevel Inheritance

  • A class is derived from a class which is also derived from another class.

    • class A:
    pass

    class B(A):
    pass

    class C(B):
    pass

    Hierarchical Inheritance

  • Multiple child classes inherit from a single parent class.

    • class A:
    pass

    class B(A):
    pass

    class C(A):
    pass

    Hybrid Inheritance

  • Combination of more than one type of inheritance.

    • `super()` Function

  • Used to call the constructor or methods of the **parent class**.

    • class Parent:
    def __init__(self):
    print("Parent constructor")

    class Child(Parent):
    def __init__(self):
    super().__init__()
    print("Child constructor")

    Method Overriding

  • Child class can override a method of the parent class with its own version.

    • class Parent:
    def show(self):
    print("Parent")

    class Child(Parent):
    def show(self):
    print("Child")

    obj = Child()
    obj.show() # Output: Child
