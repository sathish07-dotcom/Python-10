One child class inherits from one parent class. ```python class A: pass class B(A): pass ``` 2. **Multiple Inheritance** - A child class inherits from multiple parent classes. ```python class A: pass class B: pass class C(A, B): pass ``` 3. **Multilevel Inheritance** -
